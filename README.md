# Stash-Buttplug Repository

This repository hosts the **Stash-Buttplug** plugin for StashApp.

## Installation

Add this repository as a plugin source in StashApp:

1. Go to **Settings > Plugins**.
2. Click **Sources** -> **Add Source**.
3. Enter:
   - **Name**: Stash-Buttplug
   - **URL**: `https://happykinkster.github.io/stash-buttplug/index.yml`
4. install the **Stash-Buttplug** plugin from the **Available** tab.

## Plugin Details

Connects StashApp to Buttplug.io (Intiface) to control devices like the Kiiroo Keon using Funscripts synced with your scenes.

See [plugins/Stash-Buttplug/README.md](plugins/Stash-Buttplug/README.md) for usage details.
