# stashInteractive Repository

This repository hosts the **stashButtplug** and **stashVR** plugin for StashApp.

## Installation

Add this repository as a plugin source in StashApp:

1. Go to **Settings > Plugins**.
2. Click **Sources** -> **Add Source**.
3. Enter:
   - **Name**: stashInteractive
   - **URL**: `https://happykinkster.github.io/stashInteractive/main/index.yml`
4. install the **stashButtplug** and **stashVR** plugin from the **Available** tab.

## Plugin Details

Connects StashApp to Buttplug.io (Intiface) to control devices like the Kiiroo Keon using Funscripts synced with your scenes.

See [plugins/stashButtplug/README.md](plugins/stashButtplug/README.md) for usage details.

See [plugins/stashVR/README.md](plugins/stashVR/README.md) for usage details.
